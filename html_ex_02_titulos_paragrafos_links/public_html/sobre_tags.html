TAGS

Tags representam os elementos da linguagem de marcação.

Os elementos são as menores unidades HTML que fornecem alguma informação sobre a estruturação do conteúdo.

Tags possuem pares de abertura e fechamento que contém o nome do elemento representado, seus atributos (contidos no tag de abertura) e um conteúdo que está "envolto" pelos pares de tags.

Exemplo: <p name="paragrafo_01" > Parágrafo </p>
